  • Support Email: jbbr1990@gmail.com

Introduction

NewsTree is a responsive news and blog portal CMS script. It has a nice and attractive front end interface that is really appealing. Also the backend has a lot of awesome features that are needed for a modern and professional newspaper website. Almost all kinds of task can be done using the admin panel without having any kind or programming language knowledge.

This CMS is built using the procedural php (with PDO feature) and without any framework. For this reason, it can be customized easily by most of the developers. This CMS is strong against SQL injection, and XSS attack; and your website will be safe. Also in all pages, admin level security and invalid page url submission checking were done.

Requirements

Before starting to install our item, make sure you fulfill the following requirements:

  1. For Local Server installation, you need to have XAMPP, WAMP or any apache server with PHP and MySQL.
  2. Mod Rewrite will have to be enabled if you want to get SEO friendly URL. Examples are given below:

    If Mod Rewrite is not enabled, a page url will be like: 
    http://www.yourwebsite.com/page.php?slug=about-us
    If Mod Rewrite is enabled, a page url will be like: 
    http://www.yourwebsite.com/page/about-us
  3. You have use any live LINUX server to use our script without problem.
  4. Required PHP version in server >= 5.4.
  5. PDO must be enabled in the server.

Installation

In order to install the CMS, please follow these steps:

  1. Search for the zip file cms.zip in the main theme folders that you have downloaded.
  2. Now upload the cms.zip file into your server using FTP or Cpanel. Extract this and use. You can use our cms into root domain or subdomain.
  3. Create mysql database using cpanel or your hosting provider's system. Then import the xicia_newstree.sql file (that you got into the theme's folder) into your created database.
  4. Now you will have to edit the configuration file. Into our cms folder, you will get a folder named admin.

  5. Into admin folder you will get a file config.php. Open it with a text editor (for example: notepad++).

  6. Into the config.php file you will get the important information. If you do not setup this correctly, the script will not work. Change the marked information like the image below.
    • Time Zone: Write your proper timezone here
    • Host Name: This is your database host name or IP address
    • Database Name: Create a database name in your host (localhost or webhost) and put name here
    • Database Username: Username of the database you created
    • Database Password: Password of the database you created
    • BASE URL: This is the main website URL where you are trying to run our script. Do not forget to put the trail slash in the URL, otherwise the script will not work.

Dashboard

In the dashboard, you will be able to get all the total data for news, categories, users, photos, videos and subscribers. From the top right corner menu, any user can change his profile information, password, photo etc.

User

In this CMS, unlimited users can be created under total 2 roles, one is admin and another is publisher. Here user's email address will be treated as username in this website. User photo upload is optional field. User can be setup as active, edit or delete.

Admin can do almost everything of this website, except removing Super Admin and managing the users.

Publisher can do only work with file, news, photo, video and subscriber options.

Settings

In this cms, you can change the following things in the settings:

  • Logo Image
  • Favicon Image
  • Footer About Us section, copyright section, contact or website email, number, fax and google map code
  • Contact form email address, subject and thank you message
  • Setting up how many recent news and popular news will be shown in the sidebar
  • Home page meta title, meta keywords and meta description
  • Mod Rewrite can be setup as "On" or "Off" from here.

Home Page

In home page there are some categories to show. Actually we gave the option to add unlimited number of categories here. You can setup one from total four layouts for those category sections. Also you can set order of those categories to show on the home page. If you keep the order field blank, that category item will not be shown on the home page. In below, we are telling details about the layouts:

  1. Normal 1 Column: In this layout all posts will be shown under a category in one column. Featured Image will be shown in left and text in right.
  2. Normal 2 Columns: In this layout all posts will be shown under a category in two columns. Featured Image will be shown in top and text in bottom.
  3. Normal 3 Columns: In this layout all posts will be shown under a category in three columns. Featured Image will be shown in top and text in bottom.
  4. 2 Columns (6 posts): In this layout all posts will be shown under a category in two columns. In left column only one post will be shown where featured image will be in top and text in bottom. In the right column total 5 posts will be shown in small size.

Category

When you will try to add a new category, you will have to follow these steps:

  1. In the category page, click on Add New.
  2. Put Category Name in the text field. It is mandatory.
  3. You can put category slug manually, or keep empty. If you keep this empty, slug will be automatically generated by the system.
  4. You can make the category Active or Inactive.
  5. Put the SEO information (meta title, meta keyword and meta description).

You can edit, delete, activate or deactivate category any time from the category main page.

Page

When you will try to add a new page, you will have to follow these steps:

  1. In the main page, click on Add New.
  2. Put Page Name in the text field. It is mandatory.
  3. You can put page slug manually, or keep empty. If you keep this empty, slug will be automatically generated by the system.
  4. Place the Page Content into the content editor.
  5. Select your desired Page Layout from the drop down.
  6. You can make the page Active or Inactive.
  7. Put the SEO information (meta title, meta keyword and meta description).

You can edit, delete, activate or deactivate category any time from the category main page.

Menu

You can add category item, page item, or any other manual link as the menu in this cms. When you will add category or page item as your menu item, select the desired category or page from the dropdown, then give them a order and select the parent for them. If you want to setup that menu as main menu item, select No Parent option from the dropdown, otherwise select another menu as parent. In Other Menu section, you can setup an external link or manual link and name as menu item. In this case, you will get an extra URL field to put url manually.

News

When you will try to add a news, you will have to follow these steps:

  1. In the main news page, click on Add New.
  2. Put News Title in the text field. It is mandatory.
  3. Put News Content in the text editor. It is mandatory.
  4. You can select a date for News Publish Date from the datepicker popup. You can give it a future date if you want to publish that news in a future date.
  5. Featured Photo is optional. If you put no photo a placeholder photo will be shown in the front end for that news.
  6. You can select multiple categories for a news putting tick mark beside the categories.
  7. You can make the page Published or Unpublished.
  8. News Source is optional.
  9. Publisher is optional. But if you keep it empty, the logged in user will be treated as the publisher of this news.
  10. Put the SEO information (meta title, meta keyword and meta description).

You can edit, delete, publish or unpublish news any time from the news main page.

Comment

Go to the facebook developer section (https://developers.facebook.com/docs/plugins/comments/) to get your comment codes. Then just place it in the marked textarea and update it.

File Upload

When you will try to add a file, you will have to follow these steps:

  1. In the main files page, click on Add New.
  2. Put any title as File Name just to understand about the file.
  3. Upload your desired file. Only these files are allowed: jpg, jpeg, gif, png, pdf, doc, docx, ppt, pptx, xls, xlsx, zip, rar.

You can edit or delete file any time from the file main page.

Photo Category

In the main photo category page, click on Add New. Then put a category name and make it active or inactive.

Photo

When you will try to add a photo, you will have to follow these steps:

  1. In the main photos page, click on Add New.
  2. Put any caption as Photo Caption just to understand about the photo.
  3. Upload your desired photo. Only these photos are allowed: jpg, jpeg, gif, png.
  4. Select a category for this photo.

You can edit or delete photo any time from the photo main page.

Video Category

In the main video category page, click on Add New. Then put a category name and make it active or inactive.

Video

When you will try to add a video, you will have to follow these steps:

  1. In the main videos page, click on Add New.
  2. Put any title as Video Title just to understand about the video.
  3. Put iframe Code for the video you want to add. You can take the iframe code from YouTube, Vimeo or any other website who gives you iframe code.
  4. Select a category for this video.

You can edit or delete video any time from the video main page.

Advertisement

There are total three locations to put your advertisement in the front end. From the backend, go to advertisement section. You will get there total three tabs: Under Featured News, Sidebar Top and Sidebar Bottom.

You can link the advertisement to an external URL or not. Put a jpg, jpeg, png or gif photo as advertisement. Recommended size of banner for advertisement is given below:

  1. Under Featured News: Recommended size is: 1170X120. But you can use long height image here.
  2. Sidebar Top: Recommended size is: 300X300. But you can use long height image here.
  3. Sidebar Bottom: Recommended size is: 300X300. But you can use long height image here.
